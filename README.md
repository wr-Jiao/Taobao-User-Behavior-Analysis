## 📌 项目背景
本项目基于阿里巴巴天池经典数据集 `UserBehavior.csv`。与常见的抽样分析不同，本项目挑战了 **1.001 亿条（约 3.7GB）** 的全量原始日志。通过对海量行为数据的深度挖掘，还原真实的用户转化漏斗、时间活跃特征，并构建 RF 模型为精细化运营提供决策支持。

## 🚀 项目核心亮点
1. **全量级数据处理能力**：
   - 独立处理过亿行原始数据。在高性能算力（i9 处理器）支持下，坚持使用**全量分析**而非抽样，确保了分析结论能够覆盖长尾品类和极端用户行为，提升了业务洞察的精度。
2. **严谨的数据去噪逻辑**：
   - 识别并利用 **99% 分位数** 剔除购买频次异常（>14次/9天）的用户，有效过滤了爬虫、代购等噪音数据，使分析结论聚焦于 99% 的普通真实消费者。
3. **稳健的建模思维**：
   - 在构建 RF 分层模型时，针对数据分布极不均匀导致的 `qcut` 报错，通过自定义函数映射实现评分，展示了在真实复杂数据环境下解决工程问题的能力。

## 📊 核心模块与业务发现

### 1. 转化漏斗分析 (PV-Buy)

通过对 PV -> Fav/Cart -> Buy 链路的追踪，识别出整体购买转化率为 **2.26%**。分析发现加购行为（6.17%）的转化优先级远高于收藏（3.22%），是营销资源投放的重点环节。

### 2. 24小时用户活跃趋势

- **发现**：凌晨 02:00 与午间 13:00 呈现明显的转化高峰。
- **策略**：应避免在 18:00-21:00 流量低迷期进行大规模投放，建议在 13:00 高峰前 1 小时进行精准 Push。

### 3. 用户粘性与核心品类
- 筛选购买样本量 > 100 的核心类目，挖掘出复购率高达 **43.4%** 的王牌品类（如类目 ID: 3901193）。
- 整体复购率达到 **65.54%**，展现出平台极强的用户忠诚度。

### 4. RF 用户分层营销策略
基于最近购买时间(R)和购买频次(F)构建四象限模型，实现差异化运营：
- **核心价值用户**：高频且活跃，建议提供专属会员权益。
- **重要唤回用户**：曾经忠诚但近期沉默，建议发放重磅回归礼包。
- **新晋潜力用户**：刚完成首单，建议在 24h 内推送二次回购优惠券。
