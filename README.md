\## 📌 项目背景

本项目基于阿里巴巴天池经典数据集 `UserBehavior.csv`。不同于常规的抽样分析，本项目直接挑战 \*\*1.001 亿条（约 3.7GB）\*\* 的全量原始日志。通过对海量行为数据的深度挖掘，还原真实的用户转化漏斗、时间活跃特征，并构建 RF 模型为精准营销提供决策支持。



\## 🚀 项目核心亮点

1\. \*\*亿级数据工程处理\*\*：

&#x20;  - 处理过亿行原始数据，克服了单机内存压力，完整展现全量数据的分布特征。

&#x20;  - 针对 i9 高性能算力平台优化了计算逻辑，确保了全链路分析的连贯性。

2\. \*\*严谨的数据清洗 (Quantile De-noising)\*\*：

&#x20;  - 识别并利用 \*\*99% 分位数\*\* 逻辑剔除极端高频行为用户（疑似爬虫/代购），确保分析结论对 99% 的普通消费者具有普适性。

3\. \*\*业务闭环分析\*\*：

&#x20;  - 从\*\*流量转化（漏斗）\*\*到\*\*时间规律（趋势）\*\*，再到\*\*用户粘性（复购）\*\*与\*\*用户分层（RFM）\*\*，形成完整的业务分析闭环。



\## 📊 核心模块与发现



\### 1. 转化漏斗分析 (AARRR 视角)

通过对 PV-Fav/Cart-Buy 链路的追踪，发现整体购买转化率为 \*\*2.26%\*\*。加购行为（6.17%）展现出比收藏（3.22%）更强的购买信号，是转化的关键突破口。



\### 2. 用户活跃时间特征

\- \*\*黄金窗口\*\*：凌晨 02:00 与午间 13:00 呈现明显的转化高峰。

\- \*\*策略建议\*\*：应避免在 18:00-21:00 流量低迷期进行大规模投放，转而聚焦深夜与午间休息场景。



\### 3. 品类复购诊断

\- 筛选购买样本量 > 100 的核心类目，挖掘出复购率高达 \*\*43.4%\*\* 的王牌品类（如类目 ID: 3901193）。



\### 4. RF 用户分层建模

基于最近购买时间 (Recency) 和购买频次 (Frequency) 构建四象限模型：

\- \*\*核心价值用户\*\*：高频且活跃，建议提供 VIP 非金钱激励。

\- \*\*重要唤回用户\*\*：曾经忠诚但近期沉默，建议发放重磅回归礼包。

\- \*\*新晋潜力用户\*\*：刚跨过首单门槛，建议在 24h 内推送二次回购券。
